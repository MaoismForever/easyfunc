org.omg.CosNaming public interface NamingContextExt extends NamingContextExtOperations, NamingContext, IDLEntity NamingContextExt is the extension of NamingContext which contains a set of name bindings in which each name is unique and is part of Interoperable Naming Service. Different names can be bound to an object in the same or different contexts at the same time. Using NamingContextExt, you can use URL-based names to bind and resolve. See CORBA COS Naming Specification.
