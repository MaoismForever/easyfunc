javax.xml.ws public interface Binding The Binding interface is the base interface for JAX-WS protocol bindings. List getHandlerChain() Gets a copy of the handler chain for a protocol binding instance. If the returned chain is modified a call to setHandlerChain is required to configure the binding instance with the new chain. void setHandlerChain(List chain) Sets the handler chain for the protocol binding instance. String getBindingID() Get the URI for this binding instance.
