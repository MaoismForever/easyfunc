org.omg.CORBA public final class INTERNAL extends SystemException This exception indicates an internal failure in an ORB, for example, if an ORB has detected corruption of its internal data structures. It contains a minor code, which gives more detailed information about what caused the exception, and a completion status. It may also contain a string describing the exception. See the section meaning of minor codes to see the minor codes for this exception. public INTERNAL() Constructs an INTERNAL exception with a default minor code of 0 and a completion state of COMPLETED_NO. public INTERNAL(String s) Constructs an INTERNAL exception with the specified detail message, a minor code of 0, and a completion state of COMPLETED_NO. public INTERNAL(int minor, CompletionStatus completed) Constructs an INTERNAL exception with the specified minor code and completion status. public INTERNAL(String s, int minor, CompletionStatus completed) Constructs an INTERNAL exception with the specified detail message, minor code, and completion status. A detail message is a String that describes this particular exception.
