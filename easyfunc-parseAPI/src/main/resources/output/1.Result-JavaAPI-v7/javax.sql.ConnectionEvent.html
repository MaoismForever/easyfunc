javax.sql public class ConnectionEvent extends EventObject An Event object that provides information about the source of a connection-related event. ConnectionEvent objects are generated when an application closes a pooled connection and when an error occurs. The ConnectionEvent object contains two kinds of information: The pooled connection closed by the application In the case of an error event, the SQLException about to be thrown to the application public ConnectionEvent(PooledConnection con) Constructs a ConnectionEvent object initialized with the given PooledConnection object. SQLException defaults to null. public ConnectionEvent(PooledConnection con, SQLException ex) Constructs a ConnectionEvent object initialized with the given PooledConnection object and SQLException object. public SQLException getSQLException() Retrieves the SQLException for this ConnectionEvent object. May be null.
