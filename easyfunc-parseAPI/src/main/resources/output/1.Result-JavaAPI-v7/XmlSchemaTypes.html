javax.xml.bind.annotation @Retention(value=RUNTIME) @Target(value=PACKAGE) public @interface XmlSchemaTypes A container for multiple @XmlSchemaType annotations. Multiple annotations of the same type are not allowed on a program element. This annotation therefore serves as a container annotation for multiple @XmlSchemaType annotations as follows: @XmlSchemaTypes({ @XmlSchemaType(...), @XmlSchemaType(...) }) The @XmlSchemaTypes annnotation can be used to define XmlSchemaType for different types at the package level.
