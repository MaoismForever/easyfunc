javax.xml.soap public class SOAPElementFactory extends Object Deprecated.  - Use javax.xml.soap.SOAPFactory for creating SOAPElements. SOAPElementFactory is a factory for XML fragments that will eventually end up in the SOAP part. These fragments can be inserted as children of the SOAPHeader or SOAPBody or SOAPEnvelope. Elements created using this factory do not have the properties of an element that lives inside a SOAP header document.
