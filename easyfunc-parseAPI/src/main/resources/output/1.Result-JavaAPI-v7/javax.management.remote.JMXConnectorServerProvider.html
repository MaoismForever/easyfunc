javax.management.remote public interface JMXConnectorServerProvider A provider for creating JMX API connector servers using a given protocol. Instances of this interface are created by JMXConnectorServerFactory as part of its newJMXConnectorServer method. JMXConnectorServer newJMXConnectorServer(JMXServiceURL serviceURL, Map environment, MBeanServer mbeanServer) throws IOException Creates a new connector server at the given address. Each successful call to this method produces a different JMXConnectorServer object.
