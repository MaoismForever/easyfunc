javax.management.timer public class TimerNotification extends Notification This class provides definitions of the notifications sent by timer MBeans. It defines a timer notification identifier which allows to retrieve a timer notification from the list of notifications of a timer MBean. The timer notifications are created and handled by the timer MBean. public TimerNotification(String type, Object source, long sequenceNumber, long timeStamp, String msg, Integer id) Creates a timer notification object. public Integer getNotificationID() Gets the identifier of this timer notification.
