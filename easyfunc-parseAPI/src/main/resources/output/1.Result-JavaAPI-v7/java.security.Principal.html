java.security public interface Principal This interface represents the abstract notion of a principal, which can be used to represent any entity, such as an individual, a corporation, and a login id. boolean equals(Object another) Compares this principal to the specified object. Returns true if the object passed in matches the principal represented by the implementation of this interface. String toString() Returns a string representation of this principal. int hashCode() Returns a hashcode for this principal. String getName() Returns the name of this principal.
