java.net public class UnknownServiceException extends IOException Thrown to indicate that an unknown service exception has occurred. Either the MIME type returned by a URL connection does not make sense, or the application is attempting to write to a read-only URL connection. public UnknownServiceException() Constructs a new UnknownServiceException with no detail message. public UnknownServiceException(String msg) Constructs a new UnknownServiceException with the specified detail message.
