org.xml.sax.helpers public class ParserAdapter extends Object implements XMLReader, DocumentHandler Adapt a SAX1 Parser as a SAX2 XMLReader. This module, both source code and documentation, is in the Public Domain, and comes with NO WARRANTY. See http://www.saxproject.org for further information. This class wraps a SAX1 Parser and makes it act as a SAX2 XMLReader, with feature, property, and Namespace support. Note that it is not possible to report skippedEntity events, since SAX1 does not make that information available. This adapter does not test for duplicate Namespace-qualified attribute names. public ParserAdapter() throws SAXException Construct a new parser adapter.
