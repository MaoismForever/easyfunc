java.lang public class IllegalAccessError extends IncompatibleClassChangeError Thrown if an application attempts to access or modify a field, or to call a method that it does not have access to. Normally, this error is caught by the compiler; this error can only occur at run time if the definition of a class has incompatibly changed. public IllegalAccessError() Constructs an IllegalAccessError with no detail message. public IllegalAccessError(String s) Constructs an IllegalAccessError with the specified detail message.
