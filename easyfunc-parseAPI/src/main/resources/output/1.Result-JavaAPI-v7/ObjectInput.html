java.io public interface ObjectInput extends DataInput, AutoCloseable ObjectInput extends the DataInput interface to include the reading of objects. DataInput includes methods for the input of primitive types, ObjectInput extends that interface to include objects, arrays, and Strings. Object readObject() throws ClassNotFoundException, IOException Read and return an object. The class that implements this interface defines where the object is "read" from. int read() throws IOException Reads a byte of data. This method will block if no input is available. int read(byte[] b) throws IOException Reads into an array of bytes. This method will block until some input is available. int read(byte[] b, int off, int len) throws IOException Reads into an array of bytes. This method will block until some input is available. long skip(long n) throws IOException Skips n bytes of input. int available() throws IOException Returns the number of bytes that can be read without blocking. void close() throws IOException Closes the input stream. Must be called to release any resources associated with the stream.
