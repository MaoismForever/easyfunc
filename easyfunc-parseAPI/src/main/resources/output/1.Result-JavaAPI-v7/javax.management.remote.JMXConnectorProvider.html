javax.management.remote public interface JMXConnectorProvider A provider for creating JMX API connector clients using a given protocol. Instances of this interface are created by JMXConnectorFactory as part of its newJMXConnector method. JMXConnector newJMXConnector(JMXServiceURL serviceURL, Map environment) throws IOException Creates a new connector client that is ready to connect to the connector server at the given address. Each successful call to this method produces a different JMXConnector object.
