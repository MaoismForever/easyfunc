java.rmi.activation public class UnknownGroupException extends ActivationException An UnknownGroupException is thrown by methods of classes and interfaces in the java.rmi.activation package when the ActivationGroupID parameter to the method is determined to be invalid, i.e., not known by the ActivationSystem. An UnknownGroupException is also thrown if the ActivationGroupID in an ActivationDesc refers to a group that is not registered with the ActivationSystem public UnknownGroupException(String s) Constructs an UnknownGroupException with the specified detail message.
