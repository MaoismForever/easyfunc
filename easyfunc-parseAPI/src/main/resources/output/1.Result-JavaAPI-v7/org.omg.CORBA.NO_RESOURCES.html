org.omg.CORBA public final class NO_RESOURCES extends SystemException Exception thrown when the ORB has encountered some general resource limitation. For example, the run time may have reached the maximum permissible number of open connections. It contains a minor code, which gives more detailed information about what caused the exception, and a completion status.
