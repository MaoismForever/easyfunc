java.awt.geom public abstract class RectangularShape extends Object implements Shape, Cloneable RectangularShape is the base class for a number of Shape objects whose geometry is defined by a rectangular frame. This class does not directly specify any specific geometry by itself, but merely provides manipulation methods inherited by a whole category of Shape objects.
