javax.xml.bind public abstract class JAXBIntrospector extends Object Provide access to JAXB xml binding data for a JAXB object. Intially, the intent of this class is to just conceptualize how a JAXB application developer can access xml binding information, independent if binding model is java to schema or schema to java. Since accessing the XML element name related to a JAXB element is a highly requested feature, demonstrate access to this binding information.
