javax.swing public class DefaultDesktopManager extends Object implements DesktopManager, Serializable This is an implementation of the DesktopManager. It currently implements the basic behaviors for managing JInternalFrames in an arbitrary parent. JInternalFrames that are not children of a JDesktop will use this component to handle their desktop-like actions.
