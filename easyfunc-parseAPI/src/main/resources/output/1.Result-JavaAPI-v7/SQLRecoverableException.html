java.sql public class SQLRecoverableException extends SQLException The subclass of SQLException thrown in situations where a previously failed operation might be able to succeed if the application performs some recovery steps and retries the entire transaction or in the case of a distributed transaction, the transaction branch. At a minimum, the recovery operation must include closing the current connection and getting a new connection. public SQLRecoverableException() Constructs a SQLRecoverableException object.
