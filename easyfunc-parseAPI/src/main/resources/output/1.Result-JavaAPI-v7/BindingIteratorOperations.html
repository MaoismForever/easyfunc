org.omg.CosNaming public interface BindingIteratorOperations The BindingIterator interface allows a client to iterate through the bindings using the next_one or next_n operations. The bindings iterator is obtained by using the list method on the NamingContext. boolean next_one(BindingHolder b) This operation returns the next binding. If there are no more bindings, false is returned. boolean next_n(int how_many, BindingListHolder bl) This operation returns at most the requested number of bindings. void destroy() This operation destroys the iterator.
