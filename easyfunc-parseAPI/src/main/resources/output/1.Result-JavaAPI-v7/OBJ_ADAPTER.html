org.omg.CORBA public final class OBJ_ADAPTER extends SystemException This exception typically indicates an administrative mismatch, for example, a server may have made an attempt to register itself with an implementation repository under a name that is already in use, or is unknown to the repository. It contains a minor code, which gives more detailed information about what caused the exception, and a completion status. It may also contain a string describing the exception.
