java.lang public class IndexOutOfBoundsException extends RuntimeException Thrown to indicate that an index of some sort (such as to an array, to a string, or to a vector) is out of range. Applications can subclass this class to indicate similar exceptions. public IndexOutOfBoundsException() Constructs an IndexOutOfBoundsException with no detail message. public IndexOutOfBoundsException(String s) Constructs an IndexOutOfBoundsException with the specified detail message.
