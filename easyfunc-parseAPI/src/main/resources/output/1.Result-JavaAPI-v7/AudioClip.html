java.applet public interface AudioClip The AudioClip interface is a simple abstraction for playing a sound clip. Multiple AudioClip items can be playing at the same time, and the resulting sound is mixed together to produce a composite. void play() Starts playing this audio clip. Each time this method is called, the clip is restarted from the beginning. void loop() Starts playing this audio clip in a loop. void stop() Stops playing this audio clip.
