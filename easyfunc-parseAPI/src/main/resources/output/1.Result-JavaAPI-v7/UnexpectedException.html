java.rmi public class UnexpectedException extends RemoteException An UnexpectedException is thrown if the client of a remote method call receives, as a result of the call, a checked exception that is not among the checked exception types declared in the throws clause of the method in the remote interface. public UnexpectedException(String s) Constructs an UnexpectedException with the specified detail message. public UnexpectedException(String s, Exception ex) Constructs a UnexpectedException with the specified detail message and nested exception.
