java.awt public class Panel extends Container implements Accessible Panel is the simplest container class. A panel provides space in which an application can attach any other component, including other panels. The default layout manager for a panel is the FlowLayout layout manager. public Panel() Creates a new panel using the default layout manager. The default layout manager for all panels is the FlowLayout class. public Panel(LayoutManager layout) Creates a new panel with the specified layout manager. public void addNotify() Creates the Panel's peer. The peer allows you to modify the appearance of the panel without changing its functionality. public AccessibleContext getAccessibleContext() Gets the AccessibleContext associated with this Panel. For panels, the AccessibleContext takes the form of an AccessibleAWTPanel. A new AccessibleAWTPanel instance is created if necessary.
