java.rmi.dgc public final class Lease extends Object implements Serializable A lease contains a unique VM identifier and a lease duration. A Lease object is used to request and grant leases to remote object references. public Lease(VMID id, long duration) Constructs a lease with a specific VMID and lease duration. The vmid may be null. public VMID getVMID() Returns the client VMID associated with the lease. public long getValue() Returns the lease duration.
