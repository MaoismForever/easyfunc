javax.jws @Retention(value=RUNTIME) @Target(value=METHOD) public @interface WebResult Customizes the mapping of the return value to a WSDL part and XML element. public abstract String name Name of return value. If the operation is rpc style and @WebResult.partName has not been specified, this is the name of the wsdl:part representing the return value. If the operation is document style or the return value maps to a header, this is the local name of the XML element representing the return value. public abstract String partName The name of the wsdl:part representing this return value. This is only used if the operation is rpc style, or if the operation is document style and the parameter style is BARE. public abstract String targetNamespace The XML namespace for the return value. Only used if the operation is document style or the return value maps to a header. If the target namespace is set to "", this represents the empty namespace. public abstract boolean header If true, the result is pulled from a message header rather then the message body.
