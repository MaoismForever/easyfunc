java.security public interface Guard This interface represents a guard, which is an object that is used to protect access to another object. This interface contains a single method, checkGuard, with a single object argument. checkGuard is invoked (by the GuardedObject getObject method) to determine whether or not to allow access to the object. void checkGuard(Object object) throws SecurityException Determines whether or not to allow access to the guarded object object. Returns silently if access is allowed. Otherwise, throws a SecurityException.
