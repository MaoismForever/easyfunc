javax.xml.ws @Target(value=TYPE) @Retention(value=RUNTIME) @Documented public @interface WebServiceClient Used to annotate a generated service interface. The information specified in this annotation is sufficient to uniquely identify a wsdl:service element inside a WSDL document. This wsdl:service element represents the Web service for which the generated service interface provides a client view. public abstract String name The local name of the Web service. public abstract String targetNamespace The namespace for the Web service. public abstract String wsdlLocation The location of the WSDL document for the service (a URL).
