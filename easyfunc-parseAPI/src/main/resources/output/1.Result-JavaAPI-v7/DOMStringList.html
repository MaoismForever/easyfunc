org.w3c.dom public interface DOMStringList The DOMStringList interface provides the abstraction of an ordered collection of DOMString values, without defining or constraining how this collection is implemented. The items in the DOMStringList are accessible via an integral index, starting from 0. See also the Document Object Model (DOM) Level 3 Core Specification. String item(int index) Returns the indexth item in the collection. If index is greater than or equal to the number of DOMStrings in the list, this returns null. int getLength() The number of DOMStrings in the list. The range of valid child node indices is 0 to length-1 inclusive. boolean contains(String str) Test if a string is part of this DOMStringList.
