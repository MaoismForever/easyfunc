javax.lang.model.element public enum NestingKind extends Enum The nesting kind of a type element. Type elements come in four varieties: top-level, member, local, and anonymous. Nesting kind is a non-standard term used here to denote this classification. Note that it is possible additional nesting kinds will be added in future versions of the platform.
