org.omg.CORBA.TypeCodePackage public final class BadKind extends UserException The exception BadKind is thrown when an inappropriate operation is invoked on a TypeCode object. For example, invoking the method discriminator_type() on an instance of TypeCode that does not represent an IDL union will cause the exception BadKind to be thrown. public BadKind() Constructs a BadKind exception with no reason message. public BadKind(String reason) Constructs a BadKind exception with the specified reason message.
