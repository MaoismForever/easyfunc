java.nio.file.attribute public interface PosixFileAttributes extends BasicFileAttributes File attributes associated with files on file systems used by operating systems that implement the Portable Operating System Interface (POSIX) family of standards. The POSIX attributes of a file are retrieved using a PosixFileAttributeView by invoking its readAttributes method. UserPrincipal owner() Returns the owner of the file. GroupPrincipal group() Returns the group owner of the file. Set permissions() Returns the permissions of the file. The file permissions are returned as a set of PosixFilePermission elements. The returned set is a copy of the file permissions and is modifiable. This allows the result to be modified and passed to the setPermissions method to update the file's permissions.
