javax.swing public class ComponentInputMap extends InputMap A ComponentInputMap is an InputMap associated with a particular JComponent. The component is automatically notified whenever the ComponentInputMap changes. ComponentInputMaps are used for WHEN_IN_FOCUSED_WINDOW bindings. public ComponentInputMap(JComponent component) Creates a ComponentInputMap associated with the specified component. public void setParent(InputMap map) Sets the parent, which must be a ComponentInputMap associated with the same component as this ComponentInputMap. public JComponent getComponent() Returns the component the InputMap was created for. public void put(KeyStroke keyStroke, Object actionMapKey) Adds a binding for keyStroke to actionMapKey. If actionMapKey is null, this removes the current binding for keyStroke. public void remove(KeyStroke key) Removes the binding for key from this object. public void clear() Removes all the mappings from this object.
