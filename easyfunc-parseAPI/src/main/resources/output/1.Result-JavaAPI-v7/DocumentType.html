org.w3c.dom public interface DocumentType extends Node Each Document has a doctype attribute whose value is either null or a DocumentType object. The DocumentType interface in the DOM Core provides an interface to the list of entities that are defined for the document, and little else because the effect of namespaces and the various XML schema efforts on DTD representation are not clearly understood as of this writing. DOM Level 3 doesn't support editing DocumentType nodes. DocumentType nodes are read-only. See also the Document Object Model (DOM) Level 3 Core Specification.
