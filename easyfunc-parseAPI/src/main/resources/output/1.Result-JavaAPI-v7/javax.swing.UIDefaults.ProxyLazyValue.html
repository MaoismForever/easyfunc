javax.swing public static class UIDefaults.ProxyLazyValue extends Object implements UIDefaults.LazyValue This class provides an implementation of LazyValue which can be used to delay loading of the Class for the instance to be created. It also avoids creation of an anonymous inner class for the LazyValue subclass.
