java.lang public class AbstractMethodError extends IncompatibleClassChangeError Thrown when an application tries to call an abstract method. Normally, this error is caught by the compiler; this error can only occur at run time if the definition of some class has incompatibly changed since the currently executing method was last compiled. public AbstractMethodError() Constructs an AbstractMethodError with no detail message. public AbstractMethodError(String s) Constructs an AbstractMethodError with the specified detail message.
