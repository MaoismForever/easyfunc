org.omg.CORBA.portable public interface ResponseHandler This interface is supplied by an ORB to a servant at invocation time and allows the servant to later retrieve an OutputStream for returning the invocation results. OutputStream createReply() Called by the servant during a method invocation. The servant should call this method to create a reply marshal buffer if no exception occurred. OutputStream createExceptionReply() Called by the servant during a method invocation. The servant should call this method to create a reply marshal buffer if a user exception occurred.
