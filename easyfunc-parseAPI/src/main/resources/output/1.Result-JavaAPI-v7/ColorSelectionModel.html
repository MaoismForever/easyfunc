javax.swing.colorchooser public interface ColorSelectionModel A model that supports selecting a Color. Color getSelectedColor() Returns the selected Color which should be non-null. void setSelectedColor(Color color) Sets the selected color to color. Note that setting the color to null is undefined and may have unpredictable results. This method fires a state changed event if it sets the current color to a new non-null color. void addChangeListener(ChangeListener listener) Adds listener as a listener to changes in the model. void removeChangeListener(ChangeListener listener) Removes listener as a listener to changes in the model.
