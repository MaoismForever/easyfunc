javax.swing protected class JScrollPane.ScrollBar extends JScrollBar implements UIResource By default JScrollPane creates scrollbars that are instances of this class. Scrollbar overrides the getUnitIncrement and getBlockIncrement methods so that, if the viewport's view is a Scrollable, the view is asked to compute these values.
