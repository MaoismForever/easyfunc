java.rmi.server public class ExportException extends RemoteException An ExportException is a RemoteException thrown if an attempt to export a remote object fails. A remote object is exported via the constructors and exportObject methods of java.rmi.server.UnicastRemoteObject and java.rmi.activation.Activatable. public ExportException(String s) Constructs an ExportException with the specified detail message. public ExportException(String s, Exception ex) Constructs an ExportException with the specified detail message and nested exception.
