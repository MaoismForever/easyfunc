javax.swing.text public abstract class EditorKit extends Object implements Cloneable, Serializable Establishes the set of things needed by a text component to be a reasonably functioning editor for some type of text content. The EditorKit acts as a factory for some kind of policy. For example, an implementation of html and rtf can be provided that is replaceable with other implementations. A kit can safely store editing state as an instance of the kit will be dedicated to a text component. New kits will normally be created by cloning a prototype kit. The kit will have it's setComponent method called to establish it's relationship with a JTextComponent. public EditorKit() Construct an EditorKit. public Object clone() Creates a copy of the editor kit.
