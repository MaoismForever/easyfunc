javax.management public interface DescriptorAccess extends DescriptorRead This interface is used to gain access to descriptors of the Descriptor class which are associated with a JMX component, i.e. MBean, MBeanInfo, MBeanAttributeInfo, MBeanNotificationInfo, MBeanOperationInfo, MBeanParameterInfo. ModelMBeans make extensive use of this interface in ModelMBeanInfo classes. void setDescriptor(Descriptor inDescriptor) Sets Descriptor (full replace).
