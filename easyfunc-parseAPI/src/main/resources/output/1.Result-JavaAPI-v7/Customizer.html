java.beans public interface Customizer A customizer class provides a complete custom GUI for customizing a target Java Bean. Each customizer should inherit from the java.awt.Component class so it can be instantiated inside an AWT dialog or panel. Each customizer should have a null constructor. void setObject(Object bean) Set the object to be customized. This method should be called only once, before the Customizer has been added to any parent AWT container. void addPropertyChangeListener(PropertyChangeListener listener) Register a listener for the PropertyChange event. The customizer should fire a PropertyChange event whenever it changes the target bean in a way that might require the displayed properties to be refreshed. void removePropertyChangeListener(PropertyChangeListener listener) Remove a listener for the PropertyChange event.
