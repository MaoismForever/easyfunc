It applies to certificate authority (CA) certificates only. public static final CRLReason AFFILIATION_CHANGED This reason indicates that the subject's name or other information has changed. public static final CRLReason SUPERSEDED This reason indicates that the certificate has been superseded. public static final CRLReason CESSATION_OF_OPERATION This reason indicates that the certificate is no longer needed. public static final CRLReason CERTIFICATE_HOLD This reason indicates that the certificate has been put on hold. public static final CRLReason UNUSED Unused reason. public static final CRLReason REMOVE_FROM_CRL This reason indicates that the certificate was previously on hold and should be removed from the CRL.
