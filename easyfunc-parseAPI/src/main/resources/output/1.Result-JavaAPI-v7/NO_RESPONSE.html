org.omg.CORBA public final class NO_RESPONSE extends SystemException This exception is raised if a client attempts to retrieve the result of a deferred synchronous call, but the response for the request is not yet available. It contains a minor code, which gives more detailed information about what caused the exception, and a completion status.
