javax.swing.tree public interface MutableTreeNode extends TreeNode Defines the requirements for a tree node object that can change -- by adding or removing child nodes, or by changing the contents of a user object stored in the node. void insert(MutableTreeNode child, int index) Adds child to the receiver at index. child will be messaged with setParent. void remove(int index) Removes the child at index from the receiver. void remove(MutableTreeNode node) Removes node from the receiver. setParent will be messaged on node. void setUserObject(Object object) Resets the user object of the receiver to object. void removeFromParent() Removes the receiver from its parent. void setParent(MutableTreeNode newParent) Sets the parent of the receiver to newParent.
