java.awt.datatransfer public interface Transferable Defines the interface for classes that can be used to provide data for a transfer operation. For information on using data transfer with Swing, see How to Use Drag and Drop and Data Transfer, a section in The Java Tutorial, for more information. DataFlavor[] getTransferDataFlavors() Returns an array of DataFlavor objects indicating the flavors the data can be provided in. The array should be ordered according to preference for providing the data (from most richly descriptive to least descriptive). boolean isDataFlavorSupported(DataFlavor flavor) Returns whether or not the specified data flavor is supported for this object. Object getTransferData(DataFlavor flavor) throws UnsupportedFlavorException, IOException Returns an object which represents the data to be transferred. The class of the object returned is defined by the representation class of the flavor.
