javax.lang.model.element public interface TypeParameterElement extends Element Represents a formal type parameter of a generic class, interface, method, or constructor element. A type parameter declares a TypeVariable. Element getGenericElement() Returns the generic class, interface, method, or constructor that is parameterized by this type parameter. List getBounds() Returns the bounds of this type parameter. These are the types given by the extends clause used to declare this type parameter. If no explicit extends clause was used, then java.lang.Object is considered to be the sole bound. Element getEnclosingElement() Returns the generic element of this type parameter.
