java.lang public class NoClassDefFoundError extends LinkageError Thrown if the Java Virtual Machine or a ClassLoader instance tries to load in the definition of a class (as part of a normal method call or as part of creating a new instance using the new expression) and no definition of the class could be found. The searched-for class definition existed when the currently executing class was compiled, but the definition can no longer be found. public NoClassDefFoundError() Constructs a NoClassDefFoundError with no detail message. public NoClassDefFoundError(String s) Constructs a NoClassDefFoundError with the specified detail message.
