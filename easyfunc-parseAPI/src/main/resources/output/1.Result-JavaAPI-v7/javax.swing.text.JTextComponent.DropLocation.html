javax.swing.text public static final class JTextComponent.DropLocation extends TransferHandler.DropLocation Represents a drop location for JTextComponents. public int getIndex() Returns the index where dropped data should be inserted into the associated component. This index represents a position between characters, as would be interpreted by a caret. public Position.Bias getBias() Returns the bias for the drop index. public String toString() Returns a string representation of this drop location.
