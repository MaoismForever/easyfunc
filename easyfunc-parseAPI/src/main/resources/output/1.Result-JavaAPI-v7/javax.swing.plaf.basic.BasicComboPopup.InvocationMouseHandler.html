javax.swing.plaf.basic protected class BasicComboPopup.InvocationMouseHandler extends MouseAdapter A listener to be registered upon the combo box (not its popup menu) to handle mouse events that affect the state of the popup menu. The main purpose of this listener is to make the popup menu appear and disappear.
