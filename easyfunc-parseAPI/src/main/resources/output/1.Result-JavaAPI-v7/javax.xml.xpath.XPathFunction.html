javax.xml.xpath public interface XPathFunction XPathFunction provides access to XPath functions. Functions are identified by QName and arity in XPath. Object evaluate(List args) throws XPathFunctionException Evaluate the function with the specified arguments. To the greatest extent possible, side-effects should be avoided in the definition of extension functions. The implementation evaluating an XPath expression is under no obligation to call extension functions in any particular order or any particular number of times.
