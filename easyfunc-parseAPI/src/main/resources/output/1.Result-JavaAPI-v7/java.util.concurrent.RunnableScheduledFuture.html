java.util.concurrent public interface RunnableScheduledFuture extends RunnableFuture, ScheduledFuture A ScheduledFuture that is Runnable. Successful execution of the run method causes completion of the Future and allows access to its results. boolean isPeriodic() Returns true if this is a periodic task. A periodic task may re-run according to some schedule. A non-periodic task can be run only once.
