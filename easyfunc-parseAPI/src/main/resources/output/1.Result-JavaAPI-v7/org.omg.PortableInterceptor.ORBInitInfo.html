org.omg.PortableInterceptor public interface ORBInitInfo extends ORBInitInfoOperations, Object, IDLEntity Passed to each ORBInitializer, allowing it to to register interceptors and perform other duties while the ORB is initializing. The ORBInitInfo object is only valid during ORB.init. If a service keeps a reference to its ORBInitInfo object and tries to use it after ORB.init returns, the object no longer exists and an OBJECT_NOT_EXIST exception shall be thrown.
