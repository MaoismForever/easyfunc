javax.swing.text public class BadLocationException extends Exception This exception is to report bad locations within a document model (that is, attempts to reference a location that doesn't exist). Warning: Serialized objects of this class will not be compatible with future Swing releases. The current serialization support is appropriate for short term storage or RMI between applications running the same version of Swing. As of 1.4, support for long term storage of all JavaBeansTM has been added to the java.beans package. Please see XMLEncoder. public BadLocationException(String s, int offs) Creates a new BadLocationException object. public int offsetRequested() Returns the offset into the document that was not legal.
