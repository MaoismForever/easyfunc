javax.net.ssl public class SSLPeerUnverifiedException extends SSLException Indicates that the peer's identity has not been verified. When the peer was not able to identify itself (for example; no certificate, the particular cipher suite being used does not support authentication, or no peer authentication was established during SSL handshaking) this exception is thrown. public SSLPeerUnverifiedException(String reason) Constructs an exception reporting that the SSL peer's identity has not been verifiied.
