java.net public interface CookiePolicy CookiePolicy implementations decide which cookies should be accepted and which should be rejected. Three pre-defined policy implementations are provided, namely ACCEPT_ALL, ACCEPT_NONE and ACCEPT_ORIGINAL_SERVER. See RFC 2965 sec. 3.3 & 7 for more detail. static final CookiePolicy ACCEPT_ALL One pre-defined policy which accepts all cookies. static final CookiePolicy ACCEPT_NONE One pre-defined policy which accepts no cookies. static final CookiePolicy ACCEPT_ORIGINAL_SERVER One pre-defined policy which only accepts cookies from original server. boolean shouldAccept(URI uri, HttpCookie cookie) Will be called to see whether or not this cookie should be accepted.
