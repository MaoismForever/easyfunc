org.omg.PortableInterceptor public interface ObjectReferenceTemplate extends ObjectReferenceFactory The object reference template. An instance of this must exist for each object adapter created in an ORB. The server_id, orb_id, and adapter_name attributes uniquely identify this template within the scope of an IMR.
