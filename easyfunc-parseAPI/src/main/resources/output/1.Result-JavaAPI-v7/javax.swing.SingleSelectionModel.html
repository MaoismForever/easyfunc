javax.swing public interface SingleSelectionModel A model that supports at most one indexed selection. int getSelectedIndex() Returns the model's selection. void setSelectedIndex(int index) Sets the model's selected index to index. Notifies any listeners if the model changes void clearSelection() Clears the selection (to -1). boolean isSelected() Returns true if the selection model currently has a selected value. void addChangeListener(ChangeListener listener) Adds listener as a listener to changes in the model. void removeChangeListener(ChangeListener listener) Removes listener as a listener to changes in the model.
