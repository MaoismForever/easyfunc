java.lang public class InstantiationError extends IncompatibleClassChangeError Thrown when an application tries to use the Java new construct to instantiate an abstract class or an interface. Normally, this error is caught by the compiler; this error can only occur at run time if the definition of a class has incompatibly changed. public InstantiationError() Constructs an InstantiationError with no detail message. public InstantiationError(String s) Constructs an InstantiationError with the specified detail message.
