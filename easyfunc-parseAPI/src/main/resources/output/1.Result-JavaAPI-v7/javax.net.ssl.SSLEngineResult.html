javax.net.ssl public class SSLEngineResult extends Object An encapsulation of the result state produced by SSLEngine I/O calls. A SSLEngine provides a means for establishing secure communication sessions between two peers. SSLEngine operations typically consume bytes from an input buffer and produce bytes in an output buffer. This class provides operational result values describing the state of the SSLEngine, including indications of what operations are needed to finish an ongoing handshake.
