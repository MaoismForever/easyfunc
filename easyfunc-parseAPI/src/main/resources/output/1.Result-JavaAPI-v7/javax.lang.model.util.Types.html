javax.lang.model.util public interface Types Utility methods for operating on types. Compatibility Note: Methods may be added to this interface in future releases of the platform. Element asElement(TypeMirror t) Returns the element corresponding to a type. The type may be a DeclaredType or TypeVariable. Returns null if the type is not one with a corresponding element. boolean isSameType(TypeMirror t1, TypeMirror t2) Tests whether two TypeMirror objects represent the same type. Caveat: if either of the arguments to this method represents a wildcard, this method will return false. As a consequence, a wildcard is not the same type as itself. This might be surprising at first, but makes sense once you consider that an example like this must be rejected by the compiler: List list = new ArrayList
