javax.swing.filechooser public abstract class FileFilter extends Object FileFilter is an abstract class used by JFileChooser for filtering the set of files shown to the user. See FileNameExtensionFilter for an implementation that filters using the file name extension. A FileFilter can be set on a JFileChooser to keep unwanted files from appearing in the directory listing. For an example implementation of a simple file filter, see yourJDK/demo/jfc/FileChooserDemo/ExampleFileFilter.java. For more information and examples see How to Use File Choosers, a section in The Java Tutorial. public FileFilter() null public abstract boolean accept(File f) Whether the given file is accepted by this filter. public abstract String getDescription() The description of this filter.
