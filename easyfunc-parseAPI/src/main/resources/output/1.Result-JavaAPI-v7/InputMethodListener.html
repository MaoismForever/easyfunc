java.awt.event public interface InputMethodListener extends EventListener The listener interface for receiving input method events. A text editing component has to install an input method event listener in order to work with input methods. The text editing component also has to provide an instance of InputMethodRequests. void inputMethodTextChanged(InputMethodEvent event) Invoked when the text entered through an input method has changed. void caretPositionChanged(InputMethodEvent event) Invoked when the caret within composed text has changed.
