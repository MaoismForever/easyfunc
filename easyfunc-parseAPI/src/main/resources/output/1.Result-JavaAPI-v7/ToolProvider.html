javax.tools public class ToolProvider extends Object Provides methods for locating tool providers, for example, providers of compilers. This class complements the functionality of ServiceLoader. public static JavaCompiler getSystemJavaCompiler() Gets the Java™ programming language compiler provided with this platform. public static ClassLoader getSystemToolClassLoader() Returns the class loader for tools provided with this platform. This does not include user-installed tools. Use the service provider mechanism for locating user installed tools.
