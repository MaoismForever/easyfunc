javax.management.loading public class PrivateMLet extends MLet implements PrivateClassLoader An MLet that is not added to the ClassLoaderRepository. This class acts exactly like its parent class, MLet, with one exception. When a PrivateMLet is registered in an MBean server, it is not added to that MBean server's ClassLoaderRepository.
