org.w3c.dom public interface DOMImplementationList The DOMImplementationList interface provides the abstraction of an ordered collection of DOM implementations, without defining or constraining how this collection is implemented. The items in the DOMImplementationList are accessible via an integral index, starting from 0. See also the Document Object Model (DOM) Level 3 Core Specification. DOMImplementation item(int index) Returns the indexth item in the collection. If index is greater than or equal to the number of DOMImplementations in the list, this returns null. int getLength() The number of DOMImplementations in the list. The range of valid child node indices is 0 to length-1 inclusive.
