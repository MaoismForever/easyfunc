javax.print.event public abstract class PrintJobAdapter extends Object implements PrintJobListener An abstract adapter class for receiving print job events. The methods in this class are empty. This class exists as a convenience for creating listener objects. Extend this class to create a PrintJobEvent listener and override the methods for the events of interest.
