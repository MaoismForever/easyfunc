java.awt public static enum Window.Type extends Enum Enumeration of available window types. A window type defines the generic visual appearance and behavior of a top-level window. For example, the type may affect the kind of decorations of a decorated Frame or Dialog instance. Some platforms may not fully support a certain window type. A utility window is usually a small window such as a toolbar or a palette.
